# Setup LITReview :

1 - Create  environment : python -m venv env

2 - Install requirement.txt : pip install -r requirements.txt

3 - Run server : python manage.py runserver


# All features :

1 - Signup and Login

2 - View own feed

3 - Follow users

4 - Create new Ticket

5 - Edit own Ticket

6 - Create new Review

7 - Edit own Review

8 - View Tickets

9 - View Reviews

10 - Ask Reviews

11 - Answer reviews
